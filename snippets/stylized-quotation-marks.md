Customizes the style of inline quotation marks.

- Use the `quotes` property to customize the characters used for the opening and closing quotes of a `<q>` element.

```html
<p>
  <q>Do or do not, there is no try.</q> – Yoda
</p>
```

```css
q {
  quotes: "“" "”";
}
```
