Prints a short summary of all commits.

- Use `git log --oneline` to list a short summary of all commits.

```shell
git log --oneline
```

```shell
git log --oneline
# d540ba1ab Merge network bug fix
# 3050fc0de Fix network bug
# c191f90c7 Initial commit
```
