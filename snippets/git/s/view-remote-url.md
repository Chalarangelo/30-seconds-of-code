Displays the URL of the remote repository.

- Use `git config --get remote.origin.url` to view the URL of the remote repository.

```shell
git config --get remote.origin.url
```

```shell
git config --get remote.origin.url
# https://github.com/30-seconds/30-seconds-of-code
```
