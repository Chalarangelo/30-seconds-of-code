Converts Celsius to Fahrenheit.

- Follow the conversion formula `F = 1.8 * C + 32`.

```py
def celsius_to_fahrenheit(degrees):
  return ((degrees * 1.8) + 32)
```

```py
celsius_to_fahrenheit(180) # 356.0
```
