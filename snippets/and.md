---
title: and
tags: math,logic,beginner
---

Returns `true` if both arguments are `true`, `false` otherwise.

- Use the logical and (`&&`) operator on the two given values.

```js
const and = (a, b) => a && b;
```

```js
and(true, true); // true
and(true, false); // false
and(false, false); // false
```
