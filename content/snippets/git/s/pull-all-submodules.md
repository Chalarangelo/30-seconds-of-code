Pulls all submodules from their respective remotes.

- Use `git submodule update --recursive --remote` to pull all submodules from their respective remotes.

```shell
git submodule update --recursive --remote
```

```shell
git submodule update --recursive --remote
# Pulls all submodules from their respective remotes
```
