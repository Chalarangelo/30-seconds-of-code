Clones missing submodules and checks out commits.

- Use `git submodule update --init --recursive` to clone missing submodules and checkout commits.

```shell
git submodule update --init --recursive
```

```shell
git submodule update --init --recursive
# Clones missing submodules and checks out commits
```
